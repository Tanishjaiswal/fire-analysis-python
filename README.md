# fire-analysis-python
# 🔥 Satellite-Based Fire Analysis using Python

## 📌 Project Overview

This project analyzes satellite-based fire detection data (MODIS dataset) using Python. It focuses on understanding fire patterns, visualizing data, and predicting fire intensity using machine learning.

## 📊 Features

* Data Cleaning & Preprocessing
* Exploratory Data Analysis (EDA)
* Data Visualization (Histogram, Scatter, Heatmap, Bar, Donut Chart)
* Machine Learning (Linear Regression)

## 📂 Dataset

The dataset is obtained from NASA's MODIS satellite system via FIRMS.

🔗 https://firms.modaps.eosdis.nasa.gov/

## 🛠️ Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn

## 📈 Key Insights

* Most fires are low intensity
* Strong relationship between temperature and intensity
* Aqua satellite detects slightly more fires than Terra

## 🚀 How to Run

1. Install required libraries
2. Run the Python script
3. Visualizations will be generated
